# KIIT-AURA-Campus-Event-Management
A full-stack Django-based campus event management system with authentication, event registration, user history tracking, star-rating feedback, media handling, and a modern UI.
## About the Project
This is a web-based Campus Event Management System built using Django. It helps students and organizers create, manage, and participate in college events easily.

## 🚀 Features
- User Registration and Login  
- Create and Manage Events  
- View Available Events  
- Event Details Page  
- Simple and User-Friendly Interface  

## 🛠️ Tech Stack
- Backend: Django (Python)  
- Frontend: HTML, CSS, JavaScript  
- Database: SQLite (default Django DB)
